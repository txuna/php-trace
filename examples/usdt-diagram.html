How USDT Works in Linux

User Statically Defined Tracepoints · eBPF / uprobe instrumentation

01 Authoring 
// target.c
#include <sys/sdt.h>

int add(int a, int b) {
  DTRACE_PROBE2(
    myapp, add__entry,
    a, b);
  int ret = a + b;
  DTRACE_PROBE1(
    myapp, add__return,
    ret);
  return ret;
}
소스코드에 DTRACE_PROBE 매크로를 삽입.
컴파일 시 NOP + ELF 메타데이터로 변환.
→

03 Loading
VMA — Process Memory
0x4011a4
nop 0x90
semaphore = 0  · inactive
프로세스 시작 시 NOP 대기 상태.
Semaphore = 0: 구독자 없음.
실행 오버헤드 ≈ 0.

04 Register Probe
VMA — Process Memory
0x4011a4
int3 0xCC
semaphore = 1  · active
⚡ breakpoint(int3) interrupt
🔗 uprobe handler
▶ eBPF program → BPF Map
→ /proc/$PID/mem : NOP→int3, semaphore++
